Bob Ross' Happy Little Scatterplot


About This Section

This Chart demonstrates the relationship between different elments of Bob Ross's work. Each axis represents the number of times Mr. Ross painted a specific element in a season. The dots represent the number of times that those two elements were painted together in a given season. This allows us to see patterns of elements occuring together over the course of Mr. Ross's career. If the dots cluster along the diagonal between axes, the two elements are closely related. If the dots tend towards the edges, the elements rarely appear together.
