Happy Little Bar Graph

This chart shows the frequency of all elements in a given season. The vertical axis shows how many times a given element was included in paintings in a given season. As you move the slider, you can see how Mr. Ross's interests changed over time. When all the bars are tall, it indicates that Mr. Ross included a greater number of elements in that seasons's work. When the bars are low, it suggests that given season inculded a lower diversity of elements in each painting.
